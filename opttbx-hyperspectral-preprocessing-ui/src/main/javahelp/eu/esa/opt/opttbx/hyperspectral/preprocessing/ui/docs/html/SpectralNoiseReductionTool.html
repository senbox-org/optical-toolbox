Spectral Noise Reduction

Overview

The Spectral Noise Reduction operator reduces noise in hyperspectral data by smoothing each pixel spectrum along the spectral axis. The selected spectral bands are ordered by wavelength and filtered with a one-dimensional convolution kernel.

Only bands with a spectral wavelength greater than zero are processed. Invalid, NaN, infinite, and no-data samples are ignored during filtering.

The target product keeps the source geometry and metadata and contains the filtered spectral bands.

Parameters

Source Bands

The spectral bands to be processed. If no bands are selected, all spectral bands of the source product are used.

Filter Method

Selects the smoothing method: Savitzky-Golay, Gaussian, or Box.

  • Savitzky-Golay - preserves local spectral shape and narrow features better than simple averaging.
  • Gaussian - smooths with stronger weight near the center band and lower weight toward the edges.
  • Box - applies a simple moving average over the selected spectral window.

Kernel Size

Size of the spectral filter window. The value must be odd, at least 3, and not larger than the number of selected spectral bands.

Gaussian Sigma

Standard deviation of the Gaussian kernel. This parameter is only used for Gaussian filtering. Larger values spread the weights more broadly across neighboring bands.

Polynomial Order

Polynomial order used for Savitzky-Golay filtering. The value must be at least 0 and smaller than the kernel size.

  • 2 - robust default
  • 3 - good general-purpose choice
  • 4 or 5 - preserves more local structure, but may be less stable with small kernels

Notes

  • Smaller kernels preserve detail better but reduce noise less strongly.
  • Larger kernels smooth more strongly but may flatten narrow spectral features.
  • At spectral boundaries, the nearest edge band is reused for missing neighbors.

Recommended Starting Values

  • Savitzky-Golay: Kernel Size = 11, Polynomial Order = 3
  • Gaussian: Kernel Size = 11, Gaussian Sigma = 1.0 to 2.0
  • Box: Kernel Size = 5 to 11

Savitzky-Golay is usually the best starting point when spectral feature preservation is important. Gaussian is a good general-purpose alternative. Box is mainly useful as a simple baseline.
