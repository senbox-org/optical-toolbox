Import AVNIR-2

The AVNIR-2 reader enables the Optical Toolbox to import CEOS and GeoTIFF formatted AVNIR-2 Level-1 data products.

Note: The CEOS AVNIR-2 reader is implemented in line with the "ALOS AVNIR-2 Level 1 Product Format Descriptions Rev.G" ( ) and is capable of reading data products of Level 1A, 1B1 and 1B2.
