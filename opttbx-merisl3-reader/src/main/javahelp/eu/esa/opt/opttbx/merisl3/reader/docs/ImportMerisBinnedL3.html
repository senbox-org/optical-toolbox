Import MERIS Binned Level-3 Product

This option allows to import MERIS Binned Level-3 data products. These data products provide the Level-3 statistics mean, count, min and max for a single MERIS Level-2 variable, e.g. Chlorophyll-a on a global, 9 km, integerised sinusoidal (ISIN) grid. The statistics for each variable is provided in a single file.

For the ease of use, SNAP resamples the global ISIN grid to a global Plate Carrée grid (Lat/lon projection).

MERIS Binned Level-3 data products can be freely downloaded from the A description of the file format and the ISIN grid can also be found there.
