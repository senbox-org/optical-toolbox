NBRSWIR Algorithm Specification

NBRSWIR

The NBRSWIR operator algorithm computes the Normalized Burn Ratio SWIR from a given product.

The NBRSWIR results from the following equation:

NBRSWIR = (S2 - S1 - 0.02) / (S2 + S1 + 0.1)

, where:

The bands are:

- S1 is the SWIR 1 band
- S2 is the SWIR 2 band


NBRSWIR-Flags

Also the processor computes an additional flags band called 'nbrswir_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for NBRSWIR is NAN or is Infinite
Bit 1The computed value for NBRSWIR is less than -1 (minus one)
Bit 2The computed value for NBRSWIR is greater than 1 (one)
