MmSR705 Algorithm Specification

MmSR705

The MmSR705 operator algorithm computes the Modified Simple Ratio (705 and 445 nm) from a given product.

The MmSR705 results from the following equation:

MmSR705 = (RE2 - A) / (RE2 + A)

, where:

The bands are:

- A is the Aerosols band
- RE2 is the Red Edge 2 band
- R is the Red band


MmSR705-Flags

Also the processor computes an additional flags band called 'mmsr705_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for MmSR705 is NAN or is Infinite
Bit 1The computed value for MmSR705 is less than -1 (minus one)
Bit 2The computed value for MmSR705 is greater than 1 (one)
