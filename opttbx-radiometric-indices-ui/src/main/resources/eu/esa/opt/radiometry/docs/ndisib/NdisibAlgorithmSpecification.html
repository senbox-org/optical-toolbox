NDISIb Algorithm Specification

NDISIb

The NDISIb operator algorithm computes the Normalized Difference Impervious Surface Index Blue from a given product.

The NDISIb results from the following equation:

NDISIb = (T - (B + N + S1) / 3.0) / (T + (B + N + S1) / 3.0)

, where:

The bands are:

- B is the Blue band
- N is the NIR band
- S1 is the SWIR 1 band
- T is the Thermal band


NDISIb-Flags

Also the processor computes an additional flags band called 'ndisib_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for NDISIb is NAN or is Infinite
Bit 1The computed value for NDISIb is less than -1 (minus one)
Bit 2The computed value for NDISIb is greater than 1 (one)
