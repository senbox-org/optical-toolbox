SR555 Algorithm Specification

SR555

The SR555 operator algorithm computes the Simple Ratio (555 and 750 nm) from a given product.

The SR555 results from the following equation:

SR555 = RE2 / G

, where:

The bands are:

- G is the Green band
- RE2 is the Red Edge 2 band
- R is the Red band


SR555-Flags

Also the processor computes an additional flags band called 'sr555_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for SR555 is NAN or is Infinite
Bit 1The computed value for SR555 is less than -1 (minus one)
Bit 2The computed value for SR555 is greater than 1 (one)
