TCARIOSAVI705 Algorithm Specification

TCARIOSAVI705

The TCARIOSAVI705 operator algorithm computes the TCARI/OSAVI Ratio (705 and 750 nm) from a given product.

The TCARIOSAVI705 results from the following equation:

TCARIOSAVI705 = (3 * ((RE2 - RE1) - 0.2 * (RE2 - G) * (RE2 / RE1))) / (1.16 * (RE2 - RE1) / (RE2 + RE1 + 0.16))

, where:

The bands are:

- G is the Green band
- RE1 is the Red Edge 1 band
- RE2 is the Red Edge 2 band
- R is the Red band


TCARIOSAVI705-Flags

Also the processor computes an additional flags band called 'tcariosavi705_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for TCARIOSAVI705 is NAN or is Infinite
Bit 1The computed value for TCARIOSAVI705 is less than -1 (minus one)
Bit 2The computed value for TCARIOSAVI705 is greater than 1 (one)
