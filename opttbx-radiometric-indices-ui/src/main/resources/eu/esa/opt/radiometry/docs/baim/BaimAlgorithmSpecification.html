BAIM Algorithm Specification

BAIM

The BAIM operator algorithm computes the Burned Area Index adapted to MODIS from a given product.

The BAIM results from the following equation:

BAIM = 1.0 / pow((0.05 - N) , 2.0) + pow((0.2 - S2) , 2.0)

, where:

The bands are:

- N is the NIR band
- S2 is the SWIR 2 band


BAIM-Flags

Also the processor computes an additional flags band called 'baim_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for BAIM is NAN or is Infinite
Bit 1The computed value for BAIM is less than -1 (minus one)
Bit 2The computed value for BAIM is greater than 1 (one)
