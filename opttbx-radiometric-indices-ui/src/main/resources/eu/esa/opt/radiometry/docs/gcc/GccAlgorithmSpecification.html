GCC Algorithm Specification

GCC

The GCC operator algorithm computes the Green Chromatic Coordinate from a given product.

The GCC results from the following equation:

GCC = G / (R + G + B)

, where:

The bands are:

- B is the Blue band
- G is the Green band
- R is the Red band


GCC-Flags

Also the processor computes an additional flags band called 'gcc_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for GCC is NAN or is Infinite
Bit 1The computed value for GCC is less than -1 (minus one)
Bit 2The computed value for GCC is greater than 1 (one)
