NIRvP Algorithm Specification

NIRvP

The NIRvP operator algorithm computes the Near-Infrared Reflectance of Vegetation and Incoming PAR from a given product.

The NIRvP results from the following equation:

NIRvP = ((N - R) / (N + R)) * N * par

, where:

The bands are:

- R is the Red band
- N is the NIR band


and the coefficients are:

- par is the Photosynthetically Active Radiation coefficient

NIRvP-Flags

Also the processor computes an additional flags band called 'nirvp_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for NIRvP is NAN or is Infinite
Bit 1The computed value for NIRvP is less than -1 (minus one)
Bit 2The computed value for NIRvP is greater than 1 (one)
