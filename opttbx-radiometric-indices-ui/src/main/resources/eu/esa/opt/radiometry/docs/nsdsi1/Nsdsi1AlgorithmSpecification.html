NSDSI1 Algorithm Specification

NSDSI1

The NSDSI1 operator algorithm computes the Normalized Shortwave-Infrared Difference Bare Soil Moisture Index 1 from a given product.

The NSDSI1 results from the following equation:

NSDSI1 = (S1 - S2) / S1

, where:

The bands are:

- S1 is the SWIR 1 band
- S2 is the SWIR 2 band


NSDSI1-Flags

Also the processor computes an additional flags band called 'nsdsi1_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for NSDSI1 is NAN or is Infinite
Bit 1The computed value for NSDSI1 is less than -1 (minus one)
Bit 2The computed value for NSDSI1 is greater than 1 (one)
