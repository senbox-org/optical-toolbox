MLSWI27 Algorithm Specification

MLSWI27

The MLSWI27 operator algorithm computes the Modified Land Surface Water Index (MODIS Bands 2 and 7) from a given product.

The MLSWI27 results from the following equation:

MLSWI27 = (1.0 - N - S2) / (1.0 - N + S2)

, where:

The bands are:

- N is the NIR band
- S2 is the SWIR 2 band


MLSWI27-Flags

Also the processor computes an additional flags band called 'mlswi27_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for MLSWI27 is NAN or is Infinite
Bit 1The computed value for MLSWI27 is less than -1 (minus one)
Bit 2The computed value for MLSWI27 is greater than 1 (one)
