GVMI Algorithm Specification

GVMI

The GVMI operator algorithm computes the Global Vegetation Moisture Index from a given product.

The GVMI results from the following equation:

GVMI = ((N + 0.1) - (S2 + 0.02)) / ((N + 0.1) + (S2 + 0.02))

, where:

The bands are:

- N is the NIR band
- S2 is the SWIR 2 band


GVMI-Flags

Also the processor computes an additional flags band called 'gvmi_flags' with the following bit coding:

Bit PositionDescription
Bit 0The computed value for GVMI is NAN or is Infinite
Bit 1The computed value for GVMI is less than -1 (minus one)
Bit 2The computed value for GVMI is greater than 1 (one)
