S3 L1C SYN Tool - Mis-registration File

Mis-registration File

The so called mis-registration file is necessary for doing the co-registration based on the instrument grids of OLCI and SLSTR. This file is briefly described within the Sentinel Technical Guide [1] This file is not yet publicly available. ESA is still working on this.
Currently those files are only available on an S3MPC FTP server (3 month rolling archive). You need to be invited to get access to those files.
The files on the FTP Server are half-orbit mis-registration files. To provide files as 3-minute granules, compatible to the SLSTR and OLCI granules, is still work in progress.
