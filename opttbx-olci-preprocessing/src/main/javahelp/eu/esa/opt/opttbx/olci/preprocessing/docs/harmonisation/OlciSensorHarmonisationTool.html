OLCI Sensor Harmonisation Processor

Introduction

The OLCI Sensor Harmonisation Processor provides a 'harmonisation' of all OLCI radiance bands. It can be used to either process a camera flat-fielding or a sensor cross-harmonisation S3A to S3B or vice versa.
