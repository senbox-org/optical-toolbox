MERIS Level 1 Radiometric Processor

Introduction

The MERIS Level 1 Radiometric Processor offers the ability to preprocessor MERIS L1 data products. The processor only supports data from the second and third reprocessing. One or all of the following algorithms can be performed on the MERIS Level 1 source product:

  • SMILE Correction
  • Equalization
  • Radiometric Recalibration
  • Radiance to Reflectance Conversion
